Detailed Description

Menus provide the user with a list of actions and/or submenus. Submenus themselves are full fledged menus and so a hierarchical structure appears.

- (void) update
implementation

Enables or disables the receivers menu items. If the item has no target, an action binding is checked. If the target does not implement the menu item's action method or the binding's selector, the item is disabled. If the target responds to selector validateMenuItem: or validateUserInterfaceItem: (in that order) the return value is used.
